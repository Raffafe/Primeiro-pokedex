# Primeiro-pokedex
O objetivo desse projeto foi explorar ainda mais o uso do HTML, do CSS e do JavaScript, através do consumo de uma API REST para a criação de uma Pokédex.

## Api utilizada 
[PokéAPI](https://pokeapi.co/)
## Tecnologias utilizadas:
- HTML5
- CSS3
- JavaScript
